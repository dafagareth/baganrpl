02Data apa yang dikumpulkan

a. Data akun

  • Username
  • Password (disimpan dalam bentuk hash, tidak dalam teks asli)
  • Peran: operator atau pemilik (owner)
  • Tanggal pembuatan akun dan login terakhir

b. Data operasional yang kamu input

  • Catatan trip: tanggal berangkat, tanggal kembali, kapal yang dipakai
  • Catatan biaya: kategori, jumlah, keterangan
  • Catatan hasil tangkap: jenis ikan, berat, kondisi
  • Catatan penjualan: pembeli, berat terjual, harga per kg, tanggal jual
  • Catatan bagi hasil ABK

c. Data lokasi (GPS)

Saat kamu mengambil foto bukti tangkap atau penjualan, aplikasi merekam koordinat lat/long dan jam pengambilan, lalu menempelkannya sebagai watermark di foto. Koordinat ini juga dikirim ke layanan reverse geocoding OpenStreetMap (Nominatim) untuk menerjemahkan titik GPS jadi nama tempat (mis. "Padang Barat, Padang, Sumatera Barat").

Kami tidak melacak lokasi kamu di luar momen pengambilan foto. Tidak ada background tracking.

d. Foto

Foto bukti operasional (hasil tangkap, transaksi penjualan) disimpan di server kami dan ditampilkan ke owner dan operator terkait. Jika ada wajah orang lain (mis. ABK, pembeli) dalam foto, tanggung jawab izin foto ada pada operator yang mengambilnya. Jika kamu adalah orang yang tidak setuju difoto, hubungi kami untuk meminta foto tersebut dihapus.

e. Data teknis

  • Token notifikasi (FCM) untuk mengirim pemberitahuan push
  • Versi aplikasi, model perangkat, dan OS (untuk diagnosa bug)
  • Log akses API: waktu, endpoint yang dipanggil, status respons

03Untuk apa data dipakai

  • Operasional usaha. Supaya owner bisa melihat performa armada secara real-time.
  • Transparansi internal. Operator dan owner melihat angka dari sumber yang sama, sehingga catatan operasional konsisten.
  • Anti-pemalsuan. Watermark GPS+waktu di foto bukti membuat data lapangan tidak bisa dipalsukan dengan mudah.
  • Notifikasi. Owner dapat pemberitahuan saat operator input data baru.
  • Diagnosa & perbaikan. Log dipakai untuk memperbaiki bug, bukan untuk profil iklan.

Data kamu tidak dijual ke pihak mana pun. Tidak ada pengiklanan pihak ketiga. Tidak ada analytics komersial seperti Google Analytics atau Facebook Pixel di dalam aplikasi.

04Dasar hukum pemrosesan

Kami memproses data kamu berdasarkan dua hal (Psl. 20 UU PDP):

  • Persetujuan kamu. Diberikan saat pertama kali mendaftar dan menggunakan aplikasi.
  • Pelaksanaan kontrak. Antara kamu dan pemilik usaha bagan, di mana aplikasi ini adalah alat operasional yang disepakati.

05Pihak ketiga yang menerima data

Data kamu bisa diakses oleh pihak ketiga berikut, terbatas pada keperluan teknis:

  • Google Firebase (Firebase Cloud Messaging). Untuk pengiriman notifikasi push. Yang dikirim hanya token perangkat dan teks notifikasi singkat. Server Firebase berlokasi di luar Indonesia.
  • OpenStreetMap / Nominatim. Untuk reverse geocoding GPS jadi nama tempat. Yang dikirim hanya koordinat lat/long, tidak ada identitas kamu. Server Nominatim berlokasi di Eropa.
  • Penyedia hosting server. Server tempat aplikasi dijalankan. Saat ini berlokasi di Indonesia.

Selain di atas, data kamu tidak kami serahkan ke pihak ketiga lain, kecuali ada permintaan resmi penegak hukum yang sah secara hukum Indonesia.

06Berapa lama data disimpan

  • Data akun. Selama akunmu aktif. Setelah kamu menghapus akun, data identitas dihapus dalam 30 hari.
  • Data operasional (trip, biaya, tangkap, penjualan). Disimpan selama-lamanya selama dibutuhkan untuk laporan keuangan dan audit. Aturan pembukuan di Indonesia mewajibkan penyimpanan sampai 10 tahun.
  • Foto bukti. Disimpan selama trip terkait belum dihapus atau diarsipkan.
  • Log teknis. Disimpan paling lama 90 hari.

07Hak-hak kamu

Sesuai Pasal 5–13 UU PDP, kamu punya hak untuk:

  • Mendapatkan informasi yang jelas tentang data kamu (hak akses).
  • Memperbaiki data yang salah (hak koreksi).
  • Meminta data kamu dihapus (hak penghapusan).
  • Menarik kembali persetujuan yang sudah kamu berikan.
  • Memperoleh salinan data kamu dalam format yang bisa dibaca komputer (hak portabilitas).
  • Mengajukan keberatan atas pemrosesan data.
  • Menyampaikan pengaduan ke lembaga pengawas pelindungan data.

Cara menggunakan hak-hak di atas: kirim email ke dafagareth@gmail.com dengan subjek [Privasi] Permintaan Hak dan sebutkan hak yang ingin kamu pakai.

Catatan penting. Permintaan penghapusan data tidak bisa dilakukan untuk catatan transaksi yang sudah masuk laporan keuangan terkunci, karena itu termasuk kewajiban pembukuan menurut hukum perpajakan Indonesia. Yang bisa dihapus: data identitas pribadi, foto, dan catatan yang belum terkunci.

08Keamanan data

Kami pakai langkah pengamanan teknis standar: password di-hash sebelum disimpan, koneksi ke server pakai HTTPS, akses ke database dibatasi via akun bertingkat. Token autentikasi (JWT) punya masa berlaku terbatas.

Tapi: tidak ada sistem yang 100% aman. Kalau terjadi kebocoran data yang berdampak pada kamu, kami akan memberi tahu paling lambat 3×24 jam sejak kebocoran diketahui (sesuai Pasal 46 UU PDP).
